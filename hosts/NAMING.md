# Hardware Naming Scheme Reference

## Dev Machines
**Theme:** Fictional Engineers/Builders/Leaders from Sci-Fi

**Scope:** Development workstations, coding machines, design computers

**Reasoning:** The brilliant minds who design, build, and lead - these represent the creative and problem-solving work that happens on dev machines. Each name honors a fictional engineer or inventor who pushed boundaries and created impossible technology in their universes.

## Usage Notes

- Keep names lowercase for consistency in system hostnames
- Names must be at least 3 characters long
- Maximum three dashes per name (allowing up to 4 words, e.g., infinite-improbability-drive)
- Prefer using dashes to make references more explicit rather than shorter, more generic names
- Each category represents a different tier of equipment with distinct purposes
- Names should be immediately recognizable to fans of the source material
- All names reference specific fictional universes - no generic terms
